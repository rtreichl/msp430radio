Handler to select the current station.

Following steps are done to get the current selected station:
(1)Update the current station according to the encoder count
(2)If the button was pressed for a short time-> actualize current station
(3)Update the display

Parameter
<encoder_left_button>[in] state of the left encoder button
<encoder_left_count>[in] value of the left encoder
<encoder_right_button>[in] state of the right encoder button
<encoder_right_count>[in] value of the right encoder
<entry_num>[in] the entry in the menu
